# MouseSensitivityTweak

A Simple CLI to Set Mouse Sensitivity(speed)

#### Usage

    MouseSensitivityTweak <Speed>
    Ex:
    MouseSensitivityTweak 20  // Min Speed: 1, Max Speed: 20
